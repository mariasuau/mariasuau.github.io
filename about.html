Médico de familia que vive en Málaga, España.

También soy una inspirada chef, Deutschschülerin y orgullosa madre.

Pantumaca es un blog de recetas del día a día. Son recetas diferentes, inventadas, familiares, especiales para dietas, golosas... y sobre todo, fáciles y asequibles.

Este blog está alojado en GitHub usando Jekyll como sistema de contenidos. Las entradas relacionadas se obtienen con el plugin related_posts. Todas las entradas están escritas en Markdown con el editor Byword para iOS y OS X.
